Class ImplicitMSALAuthenticationProvider

deprecated

Use of ImplicitMSALAuthenticationProvider, that is, using the implicit authorization flow is not recommended. Use the TokenCredentialAuthenticationProvider with azure/identity library or a CustomAuthenticationProvider with msal-browser library instead.
